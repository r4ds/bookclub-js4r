1.4 Some code

Plotly example

  • Run the following
  • Investigate the plot
  • Look at the HTML in DevTools (id=“htmlwidget-”)
  • See where the htmlwidget script is read in
suppressPackageStartupMessages({
  library(plotly)
})

plot_ly(
  diamonds,
  x = ~cut,
  color = ~clarity,
  type = "histogram",
  width = "100%"
)
