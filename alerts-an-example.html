11.3 Alerts, an example

Goals:

  • Build a Shiny application using the jBox JavaScript library.
  • Such application will display a notification at the click of an actionButton and will tell the server when such notification is closed.
// Type into the browser console
alert("Alert message")

11.3.1 Explore

Let’s load jBox into a Shiny app, in order to play around with its features.

library(shiny)

ui <- fluidPage(
  tags$head(
    tags$script(
    # Load jBox's script
      src = paste0(
        "https://cdn.jsdelivr.net/gh/StephanWagner/",
        "jBox@v1.2.0/dist/jBox.all.min.js"
      )
    ),
    # Load jBox's styling
    tags$link(
      rel = "stylesheet",
      href = paste0(
        "https://cdn.jsdelivr.net/gh/StephanWagner/",
        "jBox@v1.2.0/dist/jBox.all.min.css"
      )
    )
  ),
  # Create a jBox alert
  tags$script("
    new jBox('Notice', {
      content: 'Hurray! A notice!'
    });"
  ),
  # Add a title and set its HTML element identifier 
  h1("Exploring jBox", id = "title")
)

server <- function(input, output) {}

shinyApp(ui, server)

Be careful when searching for jBox’s documentation, do not Google jBox; type jBox js instead.

A couple more jBox features:

// Type into the browser console
new jBox(
  // Set plugin: Tooltip, Modal, Notice, ... 
  'Tooltip', 
  {
    attach: '#title',
    trigger: 'click',
    onCreated: function () {
      this.setContent('This is my jBox!');
    },
    onOpen: function () {
      new jBox('Notice', {content: 'Here I come!', color: 'green'});
    },
    onClose: function () {
      new jBox('Notice', {content: 'See ya later!', color: 'red'});
    }
  }
);

11.3.2 From R to JavaScript

Goal: Dynamically set the message in the alert created with jBox.

Data can be sent (through the WebSocket) from the server to the client,
via R’s session object and its sendCustomMessage method/function.

Let’s update the server function of our previous app

server <- function(input, output, session) {
  session$sendCustomMessage(
    # Set the identifier to send-notice,
    # for JavaScript to "catch"
    type = "send-notice", 
    # Data to send to JavaScript
    message = "Hi there!"
  )
}

JavaScript’s Shiny object and its method addCustomMessageHandler provides to JavaScript the instructions for when data is sent from the server, with a certain type/identifier.

tags$script(
  "Shiny.addCustomMessageHandler(
    type = 'send-notice', 
    function (message) {
      new jBox('Notice', {
        content: message
      });
    }
  );"
)

Similar example:

library(shiny)

ui <- fluidPage(
  tags$head(
    tags$script(
      src = paste0(
        "https://cdn.jsdelivr.net/gh/StephanWagner/",
        "jBox@v1.2.0/dist/jBox.all.min.js"
      )
    ),
    tags$link(
      rel = "stylesheet",
      href = paste0(
        "https://cdn.jsdelivr.net/gh/StephanWagner/",
        "jBox@v1.2.0/dist/jBox.all.min.css"
      )
    )
  ),
  textInput("msg", "notice"),
  actionButton("notify", "Notify myself"),
  tags$script(
    "Shiny.addCustomMessageHandler(
      type = 'send-notice', 
      function (message) {
        new jBox('Notice', {
          content: message
        });
        // How was the message data converted?
        console.log(message, typeof message);
      }
    );"
  )
)

server <- function(input, output, session) {
  observeEvent(input$notify, {
    session$sendCustomMessage(
      type = "send-notice",
      message = paste0(input$msg, "!!!")
    )
  })
}

shinyApp(ui, server)

11.3.3 Serialisation

  • Note that the jBox notice is configured using a JSON object:
new jBox('Notice', {
  content: 'Hurray! A notice!',
  "color": 'red'
});
  • R’s analogue for JSON object:
options <- list(
  content = 'Hurray! A notice!',
  color = 'red'
)
jsonlite::toJSON(options, pretty = TRUE, auto_unbox = TRUE)
{
  "content": "Hurray! A notice!",
  "color": "red"
}
  • Using R’s lists for sending data in JSON format:
library(shiny)

ui <- fluidPage(
  tags$head(
    tags$script(
      src = paste0(
        "https://cdn.jsdelivr.net/gh/StephanWagner/",
        "jBox@v1.2.0/dist/jBox.all.min.js"
      )
    ),
    tags$link(
      rel = "stylesheet",
      href = paste0(
        "https://cdn.jsdelivr.net/gh/StephanWagner/",
        "jBox@v1.2.0/dist/jBox.all.min.css"
      )
    ),
    tags$script(
      "Shiny.addCustomMessageHandler(
        type = 'send-notice',
        function (message) {
          // Use notice sent from the server
          new jBox('Notice', message);
        }
      );"
    )
  )
)

server <- function(input, output, session){
  # Define notice options
  notice = list(
    content = 'Hello from the server',
    color = 'black'
  )
  # Send the notice
  session$sendCustomMessage(
    type = "send-notice",
    message = notice
  )
}

shinyApp(ui, server)

11.3.4 JavaScript to R

Goal: Send data from JavaScript to R when the notice is closed.

Remember that such event (notice gets closed) can be used like this:

// Type into the browser console
new jBox(
  // Set plugin: Tooltip, Modal, Notice, ... 
  'Tooltip', 
  {
    attach: '#title',
    trigger: 'click',
    onCreated: function () {
      this.setContent('This is my jBox!');
    },
    onOpen: function () {
      new jBox('Notice', {content: 'Here I come!', color: 'green'});
    },
    onClose: function () {
      new jBox('Notice', {content: 'See ya later!', color: 'red'});
    }
  }
);
  • We’ll send data from JavaScript to the R Server, via a simplified Shiny input.
Shiny.setInputValue('inputId', inputValue)
  • Example app different than the one provided in the main book:
library(shiny)

ui <- fluidPage(
  tags$head(
    # Load jBox dependencies
    tags$script(
      src = paste0(
        "https://cdn.jsdelivr.net/gh/StephanWagner/",
        "jBox@v1.2.0/dist/jBox.all.min.js"
      )
    ),
    tags$link(
      rel = "stylesheet",
      href = paste0(
        "https://cdn.jsdelivr.net/gh/StephanWagner/",
        "jBox@v1.2.0/dist/jBox.all.min.css"
      )
    ),
    # Set custom message handler
    tags$script(
      "Shiny.addCustomMessageHandler(
        type = 'send-notice', function (message) {
          message.onClose = function () {
            Shiny.setInputValue(
              'notice_close', 
              document.getElementById('time').innerText
            );
          }
          new jBox('Notice', message);
      });"
    )
  ),
  tags$body(
    # Show time counter
    div(
      span("Time counter (in seconds): ", style = "font-weight: bold"),
      span("", id = "time")
    ),
    br(),
    # Show notice
    actionButton("show", "Show the notice"),
    # Update the time after each second
    tags$script(
      "let time = 0;
      setInterval(
        // Update second counter
        function () {
          time += 1;
          document.getElementById('time').innerText = time;
        },
        // Repeat every second
        1000
      );
    "),
  )
)

server <- function(input, output, session){
  observeEvent(input$show, {
    # Define notice options
    notice = list(
      content = "Click to send data to the R Server",
      color = 'black'
    )
    # Send the notice
    session$sendCustomMessage(
      type = "send-notice", message = notice
    )
  })

  # Print the output of the notice_close event (when fired)
  observeEvent(input$notice_close, {
    print(input$notice_close)
  })
}

shinyApp(ui, server)
